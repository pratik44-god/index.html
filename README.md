# HTML Resume Page Assignment

## Preview of Project 



## Project Overview
This project is a single-page resume website created using HTML5. The main goal of this assignment is to understand and use semantic HTML tags such as header, main, and footer.

## Header Section
I have used the header tag to create sections for:
- My name
- Professional title
- About Me section

## Main Section
I have used the main tag to write content about:
- Skills
- Experience
- Projects
- Education

## Footer Section
I have used the footer tag to create sections for:
- Contact information
- Copyright section

## How to Run the Project
1. Download or clone the repository.
2. Open the index.html file in any web browser.
3. View the resume page.
